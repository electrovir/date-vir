Function toHtmlInputString

  • toHtmlInputString(fullDate, inputType, includeSeconds): DateTimeWithSeconds

  • Converts a FullDate object into a string that is understood by type="date" or type="time" HTMLInputElement instances. Can optionally include seconds as well for type="time".

    This function ignores the timezone, the literal stored date and time numbers are simply concatenated into a string. If you wish to convert time zones, first call toNewTimezone on your FullDate object.
